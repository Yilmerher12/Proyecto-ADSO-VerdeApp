# Objetivo del Repositorio

Este repositorio tiene como propósito almacenar y organizar toda la documentación y recursos relacionados con el desarrollo del proyecto.

Aquí se encontrará información tanto de la etapa de análisis y levantamiento de requisitos, como del desarrollo e implementación del software.


# ¿Qué se encontrará en este repositorio?

Toda la documentación relacionada con el proyecto, por ejemplo:

- Requisitos funcionales y no funcionales
- Historias de usuario
- Diagramas UML

- Elicitación de requisitos
- Mockups o diseños
- Manuales
- Evidencias
- Avances del proyecto
